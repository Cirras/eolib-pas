Introduction

EOLib is a Pascal library for writing applications related to the classic MMORPG Endless Online.

It offers comprehensive support for the EO network protocol and data files, and is largely generated from the EO Protocol specification.

Compatible with Delphi (2009+) and FreePascal (3.1.1+).
